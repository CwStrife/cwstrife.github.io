# LTC Fish Browser V0.100 — Fish Info Polish Pass

## What this pass did
This pass stayed in the **catalog/content lane only** so it remains safe to hand to OpDrone without stomping on staff/category/popup work.

Targeted polish areas:
- **Clownfish**
- **Gobies & Blennies**
- **Tangs**
- **Angelfish**

## Main improvement
The goal was to kill off the last obvious "same family, same sentence" recognition notes and swap them for more species-aware wording.

### Repetitive recognition phrases removed
- Generic angelfish recognition note: **0 remaining**
- Generic goby/blenny recognition note: **0 remaining**
- Generic clownfish recognition note: **0 remaining**
- Generic tang recognition note: **0 remaining**

## Content polish details
### Clownfish
- Designer clown morphs now read differently from each other instead of sharing one generic clown line.
- Skunk clowns, maroons, Clarkii-type clowns, perculas, and broken-bar designer morphs now have more useful recognition text.
- Repeated clownfish buying guidance was also made more subgroup-aware.

### Gobies & Blennies
- Jawfish, clown gobies, shrimp gobies, sand sifters, cleaner gobies, and blennies now read more like separate subtypes.
- Recognition notes now better match burrow fish, branch perchers, substrate sifters, and perch-and-graze blennies.
- Repeated goby/blenny buying guidance was also tightened by subtype.

### Tangs
- Remaining generic tang recognition notes were replaced with species-aware descriptions for the weaker entries.
- This especially helped the back half of the tang list where several entries were still reading too generic.

### Angelfish
- Remaining generic angelfish recognition notes were replaced with more species- or genus-aware descriptions.
- Genicanthus/open-water angels, dwarf angels, and larger showpiece angels now read more distinctly.
